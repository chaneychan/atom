华东区高级销售经理，华北区高级销售经理

工作地点：上海-浦东新区，北京-海淀区

岗位职责：

  • 1、负责开发新的求职群体与"高程工作"产品的推广，持续积累客户数量，客户信息的搜集管理；
  • 2、定期对客户档案进行分析、整理提供销售分析数据；
  • 3、能策划相关的营销方案，挖掘优质的用户群体。

任职要求：

  • 1、三年以上销售经验，有一年以上互联网销售工作经验；
  • 2、具备敏锐的市场洞察力；
  • 3、能够把握和挖掘客户的需求，并提供解决方案；
  • 4、有一定的策划能力，能独立完成策划案及PPT制作；
  • 5、性格开朗、外向，主动性强，工作态度积极，热爱销售工作，能适应工作压力和敢于面对挑战；
  • 6、有较强的沟通能力，具备优秀的销售和谈判能力，以及良好的客户开拓和服务意识。

客户助理

工作地点：上海市

岗位职责：

  • 1、统计销售数据，并制作、分析相关报表；
  • 2、收集、整理销售文件及市场资料；
  • 3、根据财务报表跟踪客户回款状况；
  • 4、配合销售人员处理相关后勤工作，并完成销售经理要求的临时性工作。

任职要求：

  • 1、大专及以上学历，专业不限；
  • 2、性格活泼开朗，工作积极主动，沟通能力及责任心强，耐心、细心，能够承受工作压力；
  • 3、了解互联网行业及广告行业，有销售助理或4a／媒体工作经验者优先。
